# TraceWeave

Verifiable AI agent interactions on [ERC-8004](https://eips.ethereum.org/EIPS/eip-8004): cryptography, ChainMark steganography, optional on-chain session records. Agent outputs in a global ERC-8004 swarm trace back to the original prompter.

Local agent runtime — Outlines + llama.cpp, MCP tools, A2A delegation.
